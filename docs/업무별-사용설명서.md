# 업무 종류별 사용설명서

프로토타입 기준으로, **"이런 일을 할 때 어디서 어떻게 하면 되는지"** 정리한 가이드입니다.

---

## 1. 업무 등록 및 관리

### 새 업무 등록하기
1. 어느 화면에서든 키보드 `N` 키를 누르거나, **내 업무** 탭에서 `+ 업무 추가` 클릭
2. 모달에서 입력:
   - **제목**: 업무 이름 (예: "3월 타임세일 기획안 작성")
   - **우선순위**: 긴급 / 일반 선택
   - **상태**: 시작전 / 진행중 / 완료
   - **마감일**: 날짜 선택
   - **담당자**: 지원, 소영, 슬기 중 선택 (복수 가능)
   - **메모**: 참고사항 입력 (선택)
3. `등록` 버튼 클릭

### 업무 상태 변경하기
**방법 1 — 칸반보드에서 드래그**
- `칸반보드` 탭 → 카드를 **드래그**해서 시작전 → 진행중 → 완료 열로 이동

**방법 2 — 우클릭 메뉴**
- `내 업무` 또는 `칸반보드`에서 업무 카드를 **마우스 오른쪽 클릭** → 상태 변경 선택

### 업무에 메모 남기기
1. `내 업무` 탭에서 왼쪽 업무 목록 중 해당 업무 클릭
2. 오른쪽 상세 패널 하단의 **메모 히스토리** 확인
3. 입력란에 내용 작성 후 `메모 추가` 클릭
4. 날짜와 함께 자동 기록됨 → 진행 경과 추적에 활용

### 업무 삭제하기
- 업무 카드를 **마우스 오른쪽 클릭** → `삭제` 선택

## 2. 일정 관리

### 이번 주 내 일정 확인하기
- `내 업무` 탭 상단의 **주간 캘린더** 확인
- ◀▶ 버튼으로 다른 주 이동 가능
- 🔁 표시 = 매주 반복되는 고정 업무 (월: 주간보고, 수: 팀 회의, 금: 주간정산)

### 팀 전체 일정 확인하기
1. `캘린더` 탭으로 이동
2. **월간 보기**: 한 달 전체를 한눈에 파악
3. **주간 보기**: 시간대별 상세 일정 확인
4. 날짜 클릭 → 오른쪽 패널에 해당 날짜 일정 상세 표시

### 일정 색상 구분
| 색상 | 종류 | 예시 |
|------|------|------|
| 🔴 빨강 | 업무 마감 | "기획서 제출 D-day" |
| 🟡 노랑 | 회의 | "주간 팀 회의" |
| 🟢 초록 | 연차/휴가 | "소영 연차" |
| 🟣 보라 | 이벤트 | "타임세일 시작" |
| ⚫ 회색 | 완료됨 | 이미 끝난 일정 |

### 일정 필터링하기
- 캘린더 상단의 **종류 필터** (업무/회의/연차/이벤트) 클릭으로 토글
- **팀원 필터** (전체/지원/소영/슬기)로 특정 팀원 일정만 보기

## 3. 회의 관리

### 회의록 작성하기
1. `회의록` 탭 → `+ 회의 추가` 클릭
2. 입력 내용:
   - **제목**: 회의명 (예: "3월 3주차 주간 회의")
   - **날짜**: 회의 날짜
   - **참석자**: 참석한 팀원 선택
   - **안건**: 회의 주제 요약
   - **내용**: 회의 상세 내용 기록
   - **팀원별 체크사항**: 각 팀원이 해야 할 후속 조치 입력
3. `등록` 클릭

### 회의 후속 조치 확인하기
1. `회의록` 탭 → 왼쪽 회의 목록에서 해당 회의 클릭
2. 오른쪽 패널 하단의 **팀원별 체크사항** 확인
3. ☐ 체크박스를 클릭하면 완료 처리 (자동 저장)
4. 회의 카드의 "체크사항 2/5" 표시로 진행률 확인 가능

### 과거 회의록 찾기
- `회의록` 탭 왼쪽 목록에서 스크롤하여 확인
- 각 회의 카드에 날짜, 안건 미리보기가 표시됨

---

## 4. 공지 및 정보 공유

### 공지 올리기
1. `공지` 탭 → `+ 공지 추가` 클릭
2. 입력 내용:
   - **제목**: 공지 제목
   - **분류**: 타임세일 / 증정이벤트 / 일반공지 선택
   - **기간**: 해당되는 경우 기간 설정
   - **내용**: 상세 내용 작성
3. `등록` 클릭

### 분류별 활용 예시
| 분류 | 활용 상황 |
|------|-----------|
| 타임세일 | 이번 주 타임세일 품목, 할인율, 기간 공유 |
| 증정이벤트 | 증정 행사 안내, 조건, 기간 공유 |
| 일반공지 | 업무 변경사항, 공유할 정보, 팀 공지 |

### 안 읽은 공지 확인하기
- `공지` 탭 상단의 **안읽음** 탭 클릭 (빨간 뱃지로 개수 표시)
- 공지를 클릭하면 자동으로 읽음 처리됨

### 팀원이 공지를 읽었는지 확인하기
- 공지 상세 오른쪽 하단의 **읽음 확인** 영역 확인
- ✅ = 읽음, ❌ = 안 읽음

## 5. 메모 관리

### 개인 메모 작성하기 (업무와 무관한 메모)
1. `내 메모` 탭 → `+ 메모 추가` 클릭
2. 제목, 내용 입력 후 등록
3. 메모는 **개인 전용** — 다른 팀원에게 보이지 않음

### 업무에 진행 메모 남기기 (업무 관련 메모)
1. `내 업무` 탭 → 해당 업무 클릭
2. 오른쪽 상세 패널 하단에서 메모 입력
3. 날짜별로 히스토리가 쌓여서 진행 경과 추적 가능

### 메모와 업무 메모의 차이
| 구분 | 개인 메모 (내 메모 탭) | 업무 메모 (내 업무 탭) |
|------|----------------------|----------------------|
| 용도 | 아이디어, 메모, 체크리스트 | 업무 진행 상황 기록 |
| 위치 | 독립 탭 | 업무 상세 패널 안 |
| 공유 | 나만 보임 | 해당 업무 담당자 공유 |
| 검색 | 내 메모 탭에서 검색 가능 | 업무 제목으로 찾기 |

### 메모 검색하기
- `내 메모` 탭 상단 검색창에 키워드 입력
- 또는 헤더의 **전체 검색**(키보드 `/`)에서 검색 → 메모도 결과에 포함됨

## 6. 팀 현황 파악

### 아침에 출근했을 때 (오늘 할 일 파악)
1. 로그인 → `대시보드`가 첫 화면
2. 확인 순서:
   - **긴급 업무** 카드 숫자 확인 → 클릭하면 목록 표시
   - **오늘 할 일** 카드 확인
   - 하단 **오늘 할 일** 목록에서 구체적인 업무 확인
   - **팀원 현황** 막대로 팀원들 진행률 파악

### 다른 팀원 업무 확인하기
**방법 1 — 팀 업무 탭**
- `팀 업무` → 팀원 이름 칩 클릭 → 해당 팀원 업무만 필터링
- 테이블 보기: 마감일순/우선순위순 정렬
- 카드 보기: 팀원별 그룹 보기

**방법 2 — 칸반보드**
- `칸반보드` → 상단 팀원 필터에서 특정 팀원 선택

**방법 3 — 대시보드**
- `대시보드` 하단의 **팀원 현황** 섹션에서 완료율 확인

### 월말 성과 정리하기
1. `월별 리포트` 탭 이동
2. 확인 가능한 항목:
   - 이번 달 완료 업무 수
   - 진행중 업무 수
   - 긴급 업무 처리 건수
   - 평균 처리 기간
   - 전체 완료율
3. 팀원별 완료율 막대 차트로 비교
4. ◀▶로 이전 달 기록도 조회 가능

---

## 7. 검색 및 빠른 접근

### 전체 검색
- 키보드 `/` 키 또는 헤더 검색창 클릭
- 업무, 회의록, 공지, 메모를 **한번에 검색**
- 검색 결과 클릭 시 해당 항목으로 이동

### 키보드 단축키
| 키 | 기능 | 활용 상황 |
|----|------|-----------|
| `N` | 새 업무 등록 | 빠르게 업무 추가할 때 |
| `/` | 전체 검색 | "그거 뭐였지?" 찾을 때 |
| `1`~`9` | 탭 이동 | 마우스 없이 화면 전환 |
| `ESC` | 모달/패널 닫기 | 팝업 닫을 때 |
| `?` | 단축키 힌트 | 단축키 까먹었을 때 |

## 8. 반복 업무 관리

매월 또는 매년 정기적으로 해야 하는 업무를 **1번만 등록**하고, 해당 월에 처리했는지 체크하는 기능입니다.

### 반복 업무 등록하기
1. `반복 업무` 탭 → `+ 반복 업무 등록` 클릭
2. 입력 내용:
   - **제목**: 반복 업무명 (예: "세금계산서 발행")
   - **주기**: 매월 / 매년 선택
   - **실행일**: 매월 몇 일에 해야 하는지 (예: 5일)
   - **실행 월**: 매년 반복일 경우 몇 월인지 선택
   - **담당자**: 담당 팀원 선택
3. `등록` 클릭

### 반복 업무 체크하기
- 각 반복 업무 카드에 **최근 6개월 타임라인**이 표시됨
- 해당 월의 ☐ 체크박스를 클릭하면 ✅로 변경 (완료 처리)
- 다시 클릭하면 ☐로 되돌림
- 예시: 세금계산서 발행 → 1월 ✅, 2월 ✅, 3월 ☐ → 3월 체크 클릭 → 3월 ✅

### 반복 업무 필터링하기
- 상단 필터 칩으로 **전체 / 매월 / 매년** 구분해서 보기

### 반복 업무 삭제하기
- 반복 업무 카드를 **마우스 오른쪽 클릭** → `삭제` 선택

### 반복 업무 활용 예시
| 주기 | 업무 예시 |
|------|-----------|
| 매월 | 세금계산서 발행, 재고 실사, 월간 매출 보고서 작성 |
| 매년 | 분기별 KPI 리뷰, 연간 사업계획 수립, 연말 정산 |

## 9. 팀 목표 관리

팀의 **연간 목표**와 **월간 목표**를 작성하고 진행률을 추적하는 기능입니다.

### 연간 목표 확인/수정하기
1. `팀 목표` 탭 상단의 **연간 목표** 섹션 확인
2. 각 목표 카드에 진행률 막대가 표시됨
   - 🔴 빨강: 30% 미만 (주의)
   - 🟡 노랑: 30~70% (진행중)
   - 🟢 초록: 70% 이상 (순조로움)
3. `수정` 버튼 클릭 → 진행률(%), 메모 업데이트 → `저장`

### 월간 목표 관리하기
1. `팀 목표` 탭 하단의 **월간 목표** 섹션 확인
2. ◀▶ 버튼으로 다른 달 목표 확인
3. `+ 목표 추가` 클릭 → 이번 달 새 목표 등록
4. 기존 목표의 `수정` 버튼으로 진행률 업데이트

### 목표 활용 팁
- **월초**: 이번 달 목표 등록 + 지난달 목표 달성률 확인
- **월중**: 진행률 업데이트하면서 목표 대비 현황 파악
- **월말**: 최종 진행률 기록 → 월별 리포트와 함께 성과 정리
- 연간 목표는 분기별로 진행률을 업데이트하면 효과적

## 10. AI 어시스턴트 (채팅)

자연어로 대화하면서 **업무를 등록하거나 조회**할 수 있는 채팅 기능입니다.

### AI 채팅 시작하기
1. 화면 오른쪽 하단의 **🤖 플로팅 버튼** 클릭 (어느 화면에서든 접근 가능!)
2. 채팅 패널이 열리면 입력창에 자연어로 명령 입력
3. `전송` 버튼 클릭 또는 Enter 키
4. 다시 버튼을 클릭하면 채팅 패널 닫기 (작업 중단 없이 바로 복귀)

### 업무 등록하기 (채팅으로)
입력 예시:
- `"업무 등록: 기획서 제출"` → 새 업무 "기획서 제출" 등록
- `"긴급 업무 추가: SNS 포스팅"` → 긴급 업무로 등록
- `"내일까지 보고서 작성 등록해줘"` → 업무 등록

### 업무 조회하기 (채팅으로)
입력 예시:
- `"내 업무 뭐 있어?"` → 내가 담당하는 미완료 업무 목록
- `"긴급 업무 보여줘"` → 긴급 업무만 필터링해서 표시
- `"소영 업무 뭐 있어?"` → 소영의 업무 목록

### 회의/공지/일정 조회하기
입력 예시:
- `"최근 회의 내용 보여줘"` → 가장 최근 회의록 요약
- `"안 읽은 공지 있어?"` → 읽지 않은 공지 목록
- `"오늘 일정 알려줘"` → 오늘의 캘린더 일정
- `"이번 주 일정 보여줘"` → 이번 주 전체 일정

### 업무 완료 처리하기
입력 예시:
- `"기획서 완료 처리해줘"` → "기획서"가 포함된 업무를 찾아 완료 처리
- `"SNS 콘텐츠 완료"` → 해당 업무 완료 처리

### 도움말 보기
- `"도움"` 또는 `"도움말"` 입력 → 사용 가능한 명령어 목록 표시

### AI 채팅 활용 팁
- **어느 화면에서든** 오른쪽 하단 🤖 버튼으로 즉시 접근
- 다른 탭을 열지 않고도 현황 파악 가능
- 채팅 기록은 자동 저장됨
- 🗑️ 버튼으로 대화 기록 초기화 가능

## 업무 시나리오별 빠른 가이드

### 시나리오 A: "새 타임세일이 잡혔다"
1. `공지` 탭 → 타임세일 공지 등록 (분류: 타임세일)
2. `N` 키 → 관련 업무 등록 (예: "타임세일 상품 준비", 마감일 설정, 담당자 배정)
3. `캘린더`에서 기간 확인

### 시나리오 B: "주간 회의가 끝났다"
1. `회의록` 탭 → 회의 내용 기록 + 팀원별 체크사항 입력
2. 체크사항에서 나온 업무가 있다면 → `N` 키로 업무 등록
3. `대시보드`에서 최근 활동에 반영 확인

### 시나리오 C: "내 업무 진행 상황을 업데이트하고 싶다"
1. `내 업무` 탭 → 해당 업무 클릭
2. 오른쪽 패널에서 메모 추가 (예: "거래처 확인 완료, 수량 확정 대기중")
3. 상태 변경 필요 시 → 우클릭 또는 칸반보드에서 드래그

### 시나리오 D: "이번 달 뭘 했는지 정리해야 한다"
1. `월별 리포트` → 완료 업무 수, 팀원별 완료율 확인
2. `회의록` → 이번 달 회의 목록 훑어보기
3. `내 메모`에서 기록해둔 내용 참고

### 시나리오 E: "팀원에게 전달할 사항이 있다"
1. 전체 공유 → `공지` 탭에 등록
2. 특정 업무 관련 → 해당 업무의 메모에 기록
3. 회의에서 나온 사항 → `회의록`의 체크사항에 추가

### 시나리오 F: "매달 반복하는 업무를 빠뜨리지 않고 싶다"
1. `반복 업무` 탭 → 세금계산서, 재고실사 등 정기 업무 등록
2. 매월 초 → 반복 업무 탭에서 지난달 체크 완료 확인
3. 해당 월에 처리 후 → ☐ 클릭하여 ✅ 체크

### 시나리오 G: "월초에 팀 목표를 세우고 추적하고 싶다"
1. `팀 목표` 탭 → 월간 목표에서 `+ 목표 추가`
2. 목표명, 진행률 0%로 등록
3. 중간에 진행될 때마다 `수정` → 진행률/메모 업데이트
4. 월말에 최종 진행률 기록 → `월별 리포트`와 비교

### 시나리오 H: "손 안 대고 업무를 빠르게 등록/확인하고 싶다"
1. `AI 어시스턴트` 탭 → "내 업무 뭐 있어?" 입력
2. 현재 업무 목록 확인 후 → "업무 등록: 거래처 미팅 준비" 입력
3. 업무 끝나면 → "거래처 미팅 완료 처리해줘" 입력
4. 전부 채팅 한 곳에서 해결!
