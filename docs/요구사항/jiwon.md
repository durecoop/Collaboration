# 지원 - 요구사항

## 내가 맡은 업무 종류
- 프로젝트 리더, 전체 업무 관리

## 이런 기능이 있으면 좋겠다

### 1. 투두 리스트 (우선순위 분류)
- 업무를 **긴급 / 일보따리(일반)** 두 가지로 나눠서 등록
- 각 카테고리별로 할 일 목록을 한눈에 볼 수 있어야 함

### 2. 데드라인 표시
- 업무에 마감일(날짜)을 지정할 수 있어야 함
- 마감일이 있는 업무는 날짜가 눈에 잘 보이게 표시
- 마감일 임박 시 알림

### 3. 협업 업무 지정
- 업무를 등록할 때 팀원을 지정할 수 있어야 함
- 예: "이 일은 소영이랑 같이" 처럼 담당자를 배정
- 팀원별로 배정된 업무를 확인할 수 있어야 함

### 4. 월별 완료 업무 조회
- 처리 완료된 업무를 월 단위로 모아서 볼 수 있어야 함
- 예: "3월에 뭘 했지?" → 3월 완료 목록 확인

### 5. 업무 메모 (상세 내용 기록)
- 장기 업무나 복잡한 업무의 경우, 텍스트로 추가 내용을 기입할 수 있어야 함
- 진행 상황, 참고사항, 히스토리 등을 자유롭게 메모

### 6. 대시보드
- 로그인하면 바로 보이는 오늘의 할 일, 전체 현황
- 다른 사람이 뭐 하고 있는지 한눈에 확인

### 7. 업무 상태 관리
- 시작 전 → 진행중 → 완료 (칸반 보드 스타일)
- 드래그앤드롭으로 상태 변경

### 8. 알림
- 마감일 임박 알림
- 새 업무가 배정됐을 때 알림

### 9. 검색/필터
- 업무가 많아지면 키워드로 검색
- 담당자별, 상태별, 우선순위별 필터

### 10. 캘린더 보기
- 업무 일정을 달력 형태로 볼 수 있어야 함
- 마감일 기준으로 달력에 업무가 표시
- 한눈에 이번 달 일정 파악 가능

### 11. 이벤트 공유 게시판
- 매주 타임세일, 증정 이벤트 등 팀 공유 정보를 등록/확인
- 이벤트 일정, 내용, 담당자를 한곳에서 관리
- 팀원 누구나 등록 가능, 전체 공유

---

## 사용성 요구사항 (비개발자 팀을 위해)

- **간단한 로그인**: 이름만 클릭하면 접속 (복잡한 회원가입 X)
- **직관적인 UI**: 설명 없어도 바로 쓸 수 있는 화면
- **모바일 대응**: 폰으로도 확인할 수 있게
- **할 일 등록/수정/삭제**: 쉽고 빠르게
- **완료 체크**: 끝난 일 바로 표시
