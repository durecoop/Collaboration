## 이런 기능이 있으면 좋겠다
- 캘린더 형식으로 팀 전체 일정을 한눈에 볼 수 있는 화면
  - 월간/주간 보기 전환
  - 업무, 연차, 회의 등 일정 종류별로 색상 구분
  - 팀원별 필터링 (특정 팀원 일정만 보기)
- 캘린더에서 바로 일정 등록/수정 가능
- 개인 업무 관리 화면의 상태별 숫자(긴급/진행중/오늘할일/이달완료) 클릭 시 해당 업무 리스트 확인 가능
- 업무 탭 상단에 주간 캘린더(월~금) 표시, 요일별 업무를 한눈에 볼 수 있도록 주간 단위로 전환 가능
- 회의록 탭
  - 날짜별로 회의 내용을 등록/조회할 수 있는 기능
  - 주요 아젠다, 팀원별 체크사항을 한눈에 보기 쉽게 정리
  - 팀원별 체크사항은 개인 페이지에서도 확인 가능
- 개인 메모 기능
  - 개인 업무 페이지에서 메모를 등록/관리할 수 있는 기능
  - 메모 내용을 검색해서 쉽게 찾을 수 있는 검색 기능
- 개인 주간 캘린더에 고정 업무 기능 (팀 전체 캘린더 탭과 별개)
  - 요일별 고정 업무를 등록하면 매주 해당 요일에 자동으로 표시
  - 고정 업무 완료 여부를 체크할 수 있는 기능
- 캘린더 일정 추가 기능 개선
  - 일정 추가 시 업무 추가와 다른 전용 입력 폼 필요 (회식, 연차, 행사 등 용도)
  - 시작 시간 / 종료 시간 입력 기능
  - 장소(위치) 입력 기능
  - 메모(추가 정보) 입력 기능
  - 등록한 일정이 달력에 바로 표시되어야 함 (시간 포함)
  - 오른쪽 상세 패널에서 시간, 장소, 메모, 참여자 정보를 모두 확인 가능
  - 팀원 모두가 한눈에 볼 수 있도록 캘린더에 반영
- 업무 상태 실시간 변경 기능
  - 내 업무 탭의 업무 목록에서 시작전 / 진행중 / 완료 상태를 바로 클릭하여 변경 가능
  - 별도의 수정 화면에 들어가지 않고도 목록에서 즉시 상태가 반영되어야 함
  - 상태 변경 시 서버에 실시간 저장되어 다른 팀원도 바로 확인 가능
- 업무 수정 편의성 개선
  - 등록한 업무의 제목, 기간, 담당자, 메모 등 내용을 쉽게 수정할 수 있어야 함
  - 업무 목록에서 바로 수정 화면으로 진입할 수 있는 직관적인 UI 제공
- 캘린더 탭에 연차 등록과 일정 등록 버튼 분리
  - 연차 등록 버튼: 연차/반차(오전)/반차(오후) 종류 선택, 시작일~종료일 범위 지정, 사유 입력
  - 일정 등록 버튼: 회의, 행사/이벤트, 업무 등 일반 일정 전용 (시간, 장소, 메모 포함)
  - 각각 용도에 맞는 전용 입력 폼 제공
- 내 업무 탭에 물리적인 업무 등록 버튼 추가
  - 단축키(N) 외에도 눈에 잘 보이는 "+ 새 업무" 버튼을 업무 목록 상단에 배치
  - 클릭하면 바로 업무 등록 모달이 열리도록
