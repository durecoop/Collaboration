## 이런 기능이 있으면 좋겠다
- 캘린더 형식으로 팀 전체 일정을 한눈에 볼 수 있는 화면
  - 월간/주간 보기 전환
  - 업무, 연차, 회의 등 일정 종류별로 색상 구분
  - 팀원별 필터링 (특정 팀원 일정만 보기)
- 캘린더에서 바로 일정 등록/수정 가능
- 개인 업무 관리 화면의 상태별 숫자(긴급/진행중/오늘할일/이달완료) 클릭 시 해당 업무 리스트 확인 가능
- 업무 탭 상단에 주간 캘린더(월~금) 표시, 요일별 업무를 한눈에 볼 수 있도록 주간 단위로 전환 가능
- 회의록 탭
  - 날짜별로 회의 내용을 등록/조회할 수 있는 기능
  - 주요 아젠다, 팀원별 체크사항을 한눈에 보기 쉽게 정리
  - 팀원별 체크사항은 개인 페이지에서도 확인 가능
