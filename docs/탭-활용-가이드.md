# 탭별 활용 가이드

우리 팀 업무 관리 프로그램의 각 탭이 어떤 상황에서 유용한지 정리했습니다.

---

## 🏠 1. 대시보드

**한줄 요약**: 출근하면 가장 먼저 보는 화면

**이런 때 쓰세요**:
- 아침에 접속해서 "오늘 내가 할 일이 몇 개지?" 확인
- 긴급 업무가 있는지 빠르게 체크
- 다른 팀원이 지금 뭘 하고 있는지 한눈에 파악

**💡 팁**:
- 상단 숫자 카드(긴급/진행중/오늘할일/완료)를 **클릭하면** 해당 업무 목록이 뜹니다
- 오늘 할 일 목록을 클릭하면 바로 해당 업무 상세로 이동합니다

---

## 📋 2. 내 업무

**한줄 요약**: 나한테 배정된 업무를 관리하는 핵심 화면

**이런 때 쓰세요**:
- 새로운 업무를 등록할 때
- 내 업무 진행 상황을 업데이트할 때
- 업무에 메모(진행 내용, 참고사항)를 남길 때

**💡 팁**:
- **왼쪽**에서 업무를 클릭하면 **오른쪽**에 상세 내용이 바로 표시됩니다 (모달 안 뜸!)
- 상단 "업무 목록 / 내 메모" 탭으로 개인 메모 화면으로 전환 가능
- 주간 캘린더에서 이번 주 일정을 한눈에 확인 → ◀▶ 버튼으로 다른 주도 볼 수 있음
- 🔁 표시가 있는 건 매주 반복되는 고정 업무입니다

---

## 📊 3. 칸반보드

**한줄 요약**: 업무 상태를 드래그로 한 번에 바꾸는 화면

**이런 때 쓰세요**:
- 여러 업무의 진행 상태를 한눈에 파악할 때
- 업무를 "시작전 → 진행중 → 완료"로 빠르게 옮길 때
- 팀 전체 업무 흐름을 볼 때

**💡 팁**:
- 카드를 **드래그해서** 다른 열에 놓으면 상태가 자동 변경됩니다
- 상단 필터로 특정 팀원이나 긴급 업무만 볼 수 있습니다
- 카드에 📝 숫자가 보이면 메모가 있다는 뜻!
- 카드를 **우클릭**하면 상태 변경/삭제 메뉴가 나옵니다

---

## 📅 4. 캘린더

**한줄 요약**: 팀 전체 일정을 달력으로 보는 화면

**이런 때 쓰세요**:
- "다음 주에 뭐가 있지?" 일정 파악
- 팀원 연차/회의 일정 확인
- 특정 날짜에 어떤 업무가 몰려있는지 확인

**💡 팁**:
- 🔴업무 🟡회의 🟢연차 🟣이벤트 — 색상으로 종류를 구분합니다
- **날짜를 클릭**하면 오른쪽 패널에 그날 일정이 상세히 표시됩니다
- 상단 필터로 특정 종류나 팀원만 필터링 가능
- 월간/주간 보기 전환 가능

---

## 👥 5. 팀 업무

**한줄 요약**: "다른 사람은 뭐 하고 있지?" 확인 화면

**이런 때 쓰세요**:
- 지원이 소영이한테 "그 업무 어디까지 했어?" 물어보기 전에 직접 확인
- 팀 전체 업무를 정렬해서 우선순위 파악
- 특정 팀원의 업무량 체크

**💡 팁**:
- **테이블 보기**: 엑셀처럼 정렬 가능 → 마감일순, 우선순위순으로 보기 좋음
- **카드 보기**: 팀원별로 묶어서 보기 좋음
- 팀원 이름 칩을 클릭하면 해당 팀원 업무만 필터링

---

## 📝 6. 회의록

**한줄 요약**: 회의 내용과 후속 조치를 기록하는 화면

**이런 때 쓰세요**:
- 주간 회의 끝나고 바로 내용 정리
- "저번에 회의에서 뭐라고 했더라?" 확인
- 내가 해야 할 체크사항 확인

**💡 팁**:
- **왼쪽** 회의 목록에서 클릭하면 **오른쪽**에 상세 내용 표시
- 팀원별 체크사항의 ☐를 클릭하면 완료 체크 가능 → 자동 저장됨
- "체크사항 2/5" 같은 숫자로 진행률을 바로 확인 가능

---

## 📢 7. 공지

**한줄 요약**: 타임세일, 이벤트, 공유사항을 올리는 게시판 *(지원 니즈)*

**이런 때 쓰세요**:
- 이번 주 타임세일 정보 공유
- 증정 이벤트 안내
- 팀 전체에 알려야 할 공지사항

**💡 팁**:
- "안읽음" 탭에서 아직 안 본 공지만 모아볼 수 있음
- 공지를 클릭하면 자동으로 "읽음" 처리됨
- 오른쪽 하단에 누가 읽었는지 ✅❌로 표시됨 → 다 읽었는지 확인 가능
- 분류(타임세일/증정이벤트/일반공지)로 구분 가능

---

## 📊 8. 월별 리포트

**한줄 요약**: "이번 달에 뭘 했지?" 성과를 정리하는 화면 *(지원 니즈)*

**이런 때 쓰세요**:
- 월말에 팀 성과 정리
- 팀원별 업무 처리량 비교
- 상위 보고용 자료 참고

**💡 팁**:
- 팀원별 완료율이 막대 차트로 표시됨
- ◀▶로 이전 달 기록도 확인 가능
- 평균 처리 기간, 전체 완료율 등 통계 제공

---

## 🗒️ 9. 내 메모

**한줄 요약**: 업무와 별개로 개인 메모를 남기는 공간 *(슬기 니즈)*

**이런 때 쓰세요**:
- 거래처 미팅 내용 빠르게 메모
- 아이디어나 체크리스트 기록
- 나중에 참고할 내용 정리

**💡 팁**:
- 검색 기능으로 과거 메모를 빠르게 찾을 수 있음
- 내 업무 탭의 "내 메모" 하위 탭에서도 접근 가능
- 메모는 개인용 — 다른 팀원에게 보이지 않음

---

## 🔄 10. 반복 업무

**한줄 요약**: 매월/매년 정기 업무를 등록하고 월별 체크하는 화면 *(지원 니즈)*

**이런 때 쓰세요**:
- 세금계산서, 재고실사 등 매달 해야 하는 업무 관리
- 연간 KPI 리뷰, 사업계획 등 매년 반복되는 업무
- "지난달에 이거 했나?" 확인할 때

**💡 팁**:
- 1번만 등록하면 매월 타임라인에 자동 표시 → 해당 월만 체크하면 됨
- 최근 6개월 히스토리가 카드에 바로 보여서 누락 여부 한눈에 파악
- 전체/매월/매년 필터로 구분해서 보기 가능
- 반복 업무 카드를 우클릭하면 삭제 가능

---

## 🎯 11. 팀 목표

**한줄 요약**: 연간/월간 팀 목표를 세우고 진행률을 추적하는 화면 *(지원 니즈)*

**이런 때 쓰세요**:
- 월초에 이번 달 팀 목표 설정
- 목표 대비 진행률 중간 점검
- 연간 큰 방향 확인 및 분기별 업데이트

**💡 팁**:
- 진행률 색상: 🔴30%미만, 🟡30~70%, 🟢70%이상 → 한눈에 상태 파악
- 월간 목표는 ◀▶로 이전 달 기록도 확인 가능
- `수정` 버튼으로 진행률과 메모를 수시로 업데이트
- 월별 리포트와 함께 보면 성과 분석에 효과적

---

## 🤖 AI 어시스턴트 (플로팅 채팅)

**한줄 요약**: **어느 화면에서든** 오른쪽 하단 🤖 버튼으로 업무를 등록/조회 *(지원 니즈)*

**이런 때 쓰세요**:
- 다른 탭에서 작업 중에 빠르게 업무 등록할 때
- "내 업무 뭐 있더라?" 바로 확인할 때
- 화면 전환 없이 회의/공지/일정을 조회하고 싶을 때

**💡 팁**:
- 🤖 버튼은 **모든 화면에서** 항상 보임 → 별도 탭 이동 불필요
- `"도움"` 입력하면 사용 가능한 명령어 전체 안내
- `"업무 등록: 제목"` 형태로 즉시 업무 생성 가능
- `"기획서 완료"` 같은 자연어로 업무 상태 변경 가능
- 🗑️ 버튼으로 대화 초기화 가능
- 채팅 기록은 자동 저장되어 다음 접속 시에도 유지

---

## ⌨️ 키보드 단축키 (PC 전용)

| 키 | 기능 |
|---|---|
| `N` | 새 업무 등록 |
| `/` | 전체 검색 포커스 |
| `1`~`9` | 해당 번호 탭으로 이동 |
| `ESC` | 모달/패널 닫기 |
| `?` | 단축키 힌트 보기/숨기기 |

---

## 🖱️ 우클릭 메뉴

업무 카드(칸반, 내 업무)에서 **마우스 오른쪽 클릭** → 상태 변경, 삭제 가능
