# 업무 관리 웹 프로그램

지원, 소영, 슬기 3명이 함께 만드는 업무 관리 도구입니다.

## 처음 세팅 (1번만 하면 됨)

### 방법 1: 터미널에서 직접
```bash
git clone https://github.com/durecoop/Collaboration.git
```
> 원하는 폴더에서 실행하면 그 안에 `Collaboration` 폴더가 생깁니다.

### 방법 2: Claude한테 시키기
1. 원하는 폴더에서 터미널 열고 `claude` 입력
2. Claude에게 말하기: "https://github.com/durecoop/Collaboration.git clone 해줘"
3. 다음부터는 Collaboration 폴더에서 `claude` 열기

> 이미 받았다면 이 단계는 건너뛰세요!

## 매일 작업할 때

`Collaboration` 폴더에서 터미널을 열고:

```bash
claude
```

Claude가 켜지면 말로 시키면 됩니다:
- **시작할 때**: "pull 해줘"
- **작업 끝나면**: "커밋하고 푸시해줘"

`CLAUDE.md` 파일을 참고하세요.
