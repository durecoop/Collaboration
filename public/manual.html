🚀 시작하기

두레생협 팀 업무 관리 시스템에 처음 접속하는 방법을 안내합니다.

🌐 접속 방법

웹 브라우저(Chrome, Safari, Edge 등)에서 아래 주소로 접속하세요.

https://durecoop-team.onrender.com

PC, 태블릿, 스마트폰 모두 사용 가능합니다.

💡 참고

처음 접속 시 서버가 깨어나는 데 30초~1분 정도 걸릴 수 있습니다. 잠시 기다려주세요.

👤 로그인

별도의 아이디/비밀번호 없이, 로그인 화면에서 본인 이름을 클릭하면 바로 로그인됩니다.

팀원 목록 (7명)

이지원 임소영 박슬기 이선아 이미리 김미정 정은주
📷 로그인 화면: 초록 배경에 팀원 카드가 나열된 모습

👤 프로필 설정

로그인 후 상단의 본인 이름/아이콘을 클릭하면 프로필 설정을 할 수 있습니다.

  • 닉네임 변경: 표시되는 이름을 원하는 닉네임으로 바꿀 수 있습니다.
  • 프로필 사진 변경: 원하는 사진을 업로드하여 프로필 이미지를 설정할 수 있습니다.

📊 대시보드

로그인 직후 보이는 메인 화면입니다. 팀 전체 현황을 한눈에 파악할 수 있습니다.

🏷 상태 카드

화면 상단에 4개의 요약 카드가 표시됩니다.

카드설명
긴급긴급 우선순위 업무 개수
진행중현재 진행 중인 업무 개수
오늘 할 일오늘 마감이거나 오늘 시작인 업무 개수
완료완료된 업무 개수

각 카드를 클릭하면 해당 업무 목록으로 이동합니다.

📋 오늘 할 일 목록

대시보드 중앙에 오늘 처리해야 할 업무들이 리스트로 표시됩니다. 클릭하면 업무 상세 화면으로 이동합니다.

👥 팀원 현황

각 팀원이 현재 진행 중인 업무 수, 완료 업무 수 등을 확인할 수 있습니다.

🕔 최근 활동

팀 전체의 최근 활동 내역(업무 등록, 상태 변경, 공지 등)이 시간순으로 표시됩니다.

🤖 AI 활용 팁

대시보드에서 현황을 확인한 후, 화면 우측 하단의 🤖 AI 채팅 버튼을 눌러 "오늘 뭐 해야 해?" 라고 물어보면 오늘의 업무를 정리해 알려줍니다.

✅ 업무 관리

업무를 등록하고, 상태를 관리하고, 팀 전체 업무를 확인하는 핵심 기능입니다.

➕ 새 업무 등록

두 가지 방법으로 새 업무를 등록할 수 있습니다.

  1. 단축키 N을 누르거나, 캘린더에서 + 일정추가 버튼을 클릭합니다.
  2. 업무 등록 폼이 열리면 아래 항목을 입력합니다.

업무 등록 항목

항목설명
제목업무 이름 (필수)
우선순위긴급 또는 일보따리 (일반)
상태시작전 / 진행중 / 완료
기간 설정FROM ~ TO 날짜 지정
- 당일 날짜가 자동 입력됩니다
- +1주 버튼으로 종료일을 빠르게 설정
- 비우기 버튼으로 날짜를 지울 수 있습니다
담당 팀원로그인한 사용자가 자동 선택됩니다. 다른 팀원으로 변경 가능.
메모업무에 대한 상세 설명이나 메모
🤖 AI 도움 받기 - 메모 작성

메모 입력란 아래의 🤖 AI 도움 받기 버튼을 클릭하면, 입력한 제목을 바탕으로 AI가 메모 초안을 자동 생성합니다. 생성된 내용을 수정하여 사용하세요.

📂 내 업무 / 팀 전체 보기

업무 목록 상단의 토글 버튼으로 보기를 전환할 수 있습니다.

  • 내 업무: 나에게 배정된 업무만 표시
  • 팀 전체: 모든 팀원의 업무 표시

🔎 업무 상세

업무를 클릭하면 상세 화면이 열립니다.

  • 메모 히스토리: 이전에 작성된 메모들을 시간순으로 확인
  • 메모 추가: 새로운 메모를 추가 (AI 도움 받기 가능)
  • 상태, 우선순위, 담당자 등을 수정할 수 있습니다.

🖱 우클릭 메뉴

업무 항목 위에서 마우스 오른쪽 클릭을 하면 빠른 메뉴가 나타납니다.

  • 상태 변경: 시작전 / 진행중 / 완료로 빠르게 변경
  • 삭제: 업무를 삭제합니다

📋 칸반보드

업무를 시각적으로 관리할 수 있는 칸반보드 뷰를 제공합니다.

  • 시작전 / 진행중 / 완료 세 개의 열(column)로 구분됩니다.
  • 업무 카드를 드래그 앤 드롭하여 다른 열로 옮기면 상태가 자동으로 변경됩니다.
📷 칸반보드: 시작전 / 진행중 / 완료 3개 열에 업무 카드가 배치된 모습

📅 주간 일정

이번 주 월~금의 일정을 한눈에 확인하고 고정 업무를 관리합니다.

🗓 내 주간 일정

월요일부터 금요일까지 이번 주의 업무와 일정이 표 형태로 표시됩니다. 등록한 업무의 기간에 따라 자동으로 해당 요일에 표시됩니다.

📌 고정 업무 관리

매주 반복되는 정기 업무를 고정 업무로 등록할 수 있습니다.

  1. + 고정업무 버튼을 클릭합니다.
  2. 업무 이름과 해당 요일을 선택합니다.
  3. 등록된 고정 업무는 매주 해당 요일에 자동으로 표시됩니다.

불필요한 고정 업무는 삭제 버튼을 눌러 제거할 수 있습니다.

📅 캘린더 이벤트 표시

캘린더에 등록한 이벤트(회의, 연차 등)도 주간 일정에 함께 표시되어, 한 화면에서 전체 일정을 파악할 수 있습니다.

🤖 AI 활용 팁

AI 채팅에서 "이번 주 일정 알려줘"라고 물어보면 이번 주 전체 일정을 요약해 줍니다.

🗓 캘린더

월간/주간 뷰로 일정을 확인하고 새 일정을 추가합니다.

📄 캘린더 뷰

캘린더는 두 가지 보기 모드를 지원합니다.

  • 월간 뷰: 한 달 전체 일정을 달력 형태로 확인
  • 주간 뷰: 한 주의 일정을 상세하게 확인

📅 날짜 클릭

달력에서 날짜를 클릭하면 해당 날짜의 일정 목록을 확인할 수 있습니다.

➕ 일정 추가

+ 일정 추가 버튼을 클릭하여 새로운 일정을 등록할 수 있습니다.

🏷 종류별 필터

일정 종류에 따라 필터링하여 원하는 일정만 볼 수 있습니다.

종류용도
업무일반 업무 일정
회의팀 회의, 미팅
연차휴가, 연차
이벤트특별 행사, 이벤트
🤖 AI 활용 팁

AI 채팅에서 "다음 주 회의 일정 알려줘"라고 물어보면 등록된 회의 일정을 빠르게 확인할 수 있습니다.

📝 회의록

팀 회의 내용을 기록하고 체크사항을 관리합니다.

✍ 회의록 작성

새 회의록 작성 시 아래 항목을 입력합니다.

  • 제목 - 회의 주제
  • 날짜 - 회의 일시
  • 참석자 - 참석한 팀원 선택 (다중 선택 가능)
  • 안건 - 회의에서 다룰 주제들
  • 논의내용 - 회의에서 나온 이야기 기록
  • 체크사항 - 회의 후 해야 할 일 (To-Do)

☑ 체크사항 관리

회의록의 체크사항은 각 항목을 클릭하여 완료/미완료 상태를 토글할 수 있습니다. 회의에서 결정된 사항이 제대로 이행되고 있는지 추적하는 데 유용합니다.

🗑 회의록 삭제

더 이상 필요하지 않은 회의록은 삭제 버튼을 눌러 제거할 수 있습니다.

🤖 AI 활용 팁

AI 채팅에서 "최근 회의 내용 알려줘"라고 물어보면 최근 회의록의 핵심 내용을 요약해 줍니다.

📣 공지

팀 내 공지사항을 작성하고 읽음 상태를 관리합니다.

✍ 공지 작성

공지를 작성할 때 종류를 선택할 수 있습니다.

종류용도
타임세일타임세일 관련 안내
증정이벤트증정 이벤트 안내
일반공지기타 일반 공지사항
🤖 AI 도움 받기 - 공지 작성

공지 작성 시 🤖 AI 도움 받기 버튼을 클릭하면, 입력한 키워드나 간단한 내용을 바탕으로 AI가 체계적인 공지문을 자동으로 작성해 줍니다.

예: "3월 20일 타임세일 30% 할인" 입력 → AI가 상세한 공지 문구 생성

👀 읽음/안읽음 상태

각 공지에는 읽음/안읽음 상태가 표시됩니다. 공지를 클릭하여 확인하면 자동으로 읽음 처리됩니다.

🔍 공지 필터

상단 필터 버튼으로 원하는 공지만 볼 수 있습니다.

  • 전체: 모든 공지 표시
  • 안읽음: 아직 읽지 않은 공지만 표시
  • 읽음: 이미 읽은 공지만 표시
🤖 AI 활용 팁

AI 채팅에서 "안 읽은 공지 있어?"라고 물어보면 아직 확인하지 않은 공지 목록을 알려줍니다.

📄 내 메모

개인 메모를 작성하고 검색할 수 있는 공간입니다.

✍ 메모 작성

개인적인 메모를 자유롭게 작성할 수 있습니다. 업무 관련 메모, 아이디어, 참고 사항 등 무엇이든 기록하세요.

🗑 메모 삭제

불필요한 메모는 삭제 버튼을 눌러 제거할 수 있습니다.

🔍 메모 검색

메모가 많아졌을 때 검색 기능을 활용하여 원하는 메모를 빠르게 찾을 수 있습니다.

🤖 AI 도움 받기 - 메모 작성

메모 작성 시 🤖 AI 도움 받기 버튼을 활용하면, 간단한 키워드만 입력해도 AI가 정리된 메모 내용을 작성해 줍니다.

🔄 반복 업무

정기적으로 발생하는 업무를 관리합니다.

🕑 주기별 반복 업무

반복 업무는 주기에 따라 세 가지로 구분됩니다.

주기예시
매주주간 보고서 작성, 매주 회의
매월월간 정산, 재고 조사
매년연간 감사, 대규모 행사

☑ 타임라인 체크

반복 업무의 각 회차별로 완료 여부를 체크할 수 있습니다. 타임라인 형태로 진행 상황을 한눈에 확인하세요.

👤 담당자별 필터

특정 팀원의 반복 업무만 필터링하여 볼 수 있습니다.

🎯 팀 목표

팀의 연간/월간 목표를 설정하고 달성률을 추적합니다.

🏆 연간 목표

팀의 큰 방향성이 되는 연간 목표를 설정합니다. 각 목표에는 진행률 바가 표시되어 현재 달성 상황을 시각적으로 확인할 수 있습니다.

매출 목표 달성
65%

📅 월간 목표

연간 목표를 달성하기 위한 월간 세부 목표를 추가하고 수정할 수 있습니다. 매월 구체적인 실행 계획을 세워보세요.

🤖 AI 활용 팁

AI 채팅에서 팀 목표 달성 현황을 물어보면 전체 진행 상황을 요약해 줍니다.

🤖 AI 어시스턴트

화면 우측 하단의 🤖 버튼을 눌러 AI 채팅을 시작하세요.

💬 사용 방법

  1. 화면 우측 하단의 🤖 버튼을 클릭합니다.
  2. 채팅 창이 열리면 자유롭게 질문을 입력합니다.
  3. AI가 시스템에 등록된 데이터를 참고하여 답변합니다.

💡 질문 예시

질문AI가 해주는 것
"내 업무 알려줘"나에게 배정된 업무 목록 정리
"긴급 업무 뭐 있어?"긴급 우선순위 업무만 필터링
"최근 회의 내용"최근 회의록 핵심 내용 요약
"이번 주 일정"이번 주 전체 일정 정리
"안 읽은 공지 있어?"미확인 공지사항 알려주기
자유로운 질문Claude AI 기반 대화 (일반 질문도 가능)
💡 참고

AI 어시스턴트는 시스템에 등록된 업무, 공지, 회의록 등의 데이터를 참고하여 답변합니다. 등록되지 않은 정보에 대해서는 답변이 제한될 수 있습니다.

⌨ 단축키

💡 팁

앱 사용 중 언제든지 ? 키를 누르면 단축키 목록을 확인할 수 있습니다.

💡 AI 활용 팁 모음

AI 기능을 최대한 활용하는 방법을 정리했습니다.

✍ 메모 작성 자동화

  1. 업무 등록 시 제목만 입력합니다.
  2. 메모란 아래의 🤖 AI 도움 받기 버튼을 클릭합니다.
  3. AI가 제목을 기반으로 상세 메모 초안을 자동 생성합니다.
  4. 생성된 내용을 검토하고 필요한 부분을 수정합니다.

📣 공지 작성 자동화

  1. 공지 종류(타임세일/증정이벤트/일반공지)를 선택합니다.
  2. 핵심 키워드를 입력합니다. (예: "3/20 딸기 타임세일 30%")
  3. 🤖 AI 도움 받기 버튼을 클릭합니다.
  4. AI가 체계적이고 읽기 좋은 공지문으로 변환합니다.

🤖 AI 채팅 활용

  • 업무 현황 파악 - "오늘 뭐 해야 해?", "긴급 업무 알려줘"
  • 팀 상황 요약 - "팀 전체 업무 현황 알려줘"
  • 일정 확인 - "이번 주 일정", "다음 회의 언제야?"
  • 공지 확인 - "안 읽은 공지 있어?", "최근 공지 내용"
  • 자유 대화 - 업무 관련 질문이나 아이디어 브레인스토밍
